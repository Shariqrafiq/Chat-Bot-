## ✨ Features

- 🤖 **Local AI Processing**: Runs AI models directly in your browser
- 📱 **GitHub Mobile Optimized**: Perfect for mobile GitHub users
- 🔓 **100% Open Source**: Built with open-source AI models
- 🚀 **Multiple AI Models**: Choose from DistilGPT-2, GPT-2, Flan-T5, BERT
- 📡 **Offline Support**: Works offline once models are downloaded
- 🎨 **GitHub Dark Theme**: Beautiful GitHub-inspired design
- 💬 **Real-time Chat**: Instant responses with typing indicators
- 📲 **PWA Ready**: Install as a mobile app

## 🚀 Quick Start

### Online Demo
Visit: `https://shariq-rafiq-anim.github.io/shariq-ai-chatbot`

### Local Development
```bash
# Clone the repository
git clone https://github.com/shariq-rafiq-anim/shariq-ai-chatbot.git

# Navigate to project
cd shariq-ai-chatbot

# Open in browser
open index.html

# Or use a local server
python -m http.server 8000
# Then visit http://localhost:8000
